Shutter 0.86.1 fixes 3 bugs

Posted by Romario on Apr 7, 2010 in Blog, release | 8 comments

Shutter 0.86.1 is a bugfix release that doesn’t add any new features.

Here is the list of issues fixed:

  * Fixed bugs
    -- LP: #541751
       [Add support for Thunderbird when sending via email]
    -- LP: #543058
       [No error reporting on 'Send by mail']
    -- LP: #549732
       [print gives blank page]
* updated translations

Have fun!
