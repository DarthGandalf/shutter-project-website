Shutter meets Cloud

Posted by Romario on Aug 11, 2010 in Blog, release | 6 comments

We are happy to announce the release of Shutter 0.86.3. Shutter 0.86.3 adds support for Ubuntu One, a cloud service by Canonical. Ubuntu One allows you to publish your files easily while always keeping control over all of your files. In case are using the latest Ubuntu release (10.04 - Lucid Lynx) with its new default theme, you will also appreciate the new icons (ubuntu-mono) that come with this version of Shutter.
