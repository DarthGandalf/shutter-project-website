New bugfix release 0.89.1

Posted by Romario on Aug 16, 2012 in Allgemein, Blog, News, release | 4 comments

Shutter 0.89.1 fixes a few bugs and updates translations.

If you’ve subscribed to the Ubuntu repository, you’ll be prompted about an update being available automatically – otherwise, check the downloads page for individual packages. Updates for other distributions will be available soon.

Changelog:

  * Fixed bugs / minor improvements
    -- LP: #1007712
       [imageshack plug in fails]
    -- LP: #1020600
       [bug on save output image in cyrillic keybord layout]
    -- LP: #1021158
       [Imgur plugin doesn't work]
    -- LP: #1033931
       [libgtk2-appindicator-perl not available for lucid (Ubuntu 10.04)]
    -- LP: #1035332
       [desktop-file-validate complains]
  * updated translations

Enjoy this release!
