Independence Day!!

Posted by Romario on Aug 20, 2008 in News | 3 comments

Release of GScrot v0.50 is available!

  * implemented new backend (gscrot does not depend on scrot anymore)
  * zoom for free-hand screenshot
  * gui changes
    -- added a detachable toolbar (buttons are replaced)
  * added translations
    -- spanish
    -- catalan
